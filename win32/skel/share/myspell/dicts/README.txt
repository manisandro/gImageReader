This folder contains spelling dictionaries.

To add additional spelling dictionaries:
* Visit http://wiki.openoffice.org/wiki/Dictionaries
* Download the spelling dictionaries for the desired language
	- Caution: Download the *.zip (or *.tar.gz) archives, not the OpenOffice 3 extensions
* Extract the downloaded file, and place the extracted *.diff and *.aff file inside this folder
* If gImageReader is running, select "Redetect Languages" from the application menu, or restart the application
